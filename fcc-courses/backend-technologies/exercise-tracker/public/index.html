Exercise Tracker


Usage:

Endpoint (POST request)

{url}/api/users (creates a new user)

Example body


{
    "username": "asterki"
}

Example response


{
    "username": "ejkqwioejqwioejwqoi",
    "_id": "63d993b19cf99d16f780ae59",
    "__v": 0
}


Endpoint (GET request)

{url}/api/users/ (Gets all users)

Example response


[
    {
        "_id": "63d98ca1d772961b152628b8",
        "username": "fcc_test_16752016977",
        "__v": 0
    },
    {
        "_id": "63d98ca2d772961b152628ba",
        "username": "fcc_test_16752016979",
        "__v": 0
    },
]


Endpoint (GET request)

{url}/api/users/:id/exercises/ (Adds an Exercise) 

{
    "description": "test",
    "duration": 60,
    "date": "2016-03-23"
}

Example response


{
    "username": "asterki",
    description: "test",
    duration: 60,
    date: "Mon Jan 01 1990",
    _id: "63d98ca2d772961b152628ba"
}


Endpoint (GET request)

{url}/api/users/:id/logs/ (Gets Exercises)

Example Params


{
    "_id": "63d98ca2d772961b152628ba",
    "from": "2016-03-23",
    "to": "2016-03-24", 
    "limit": 2
}

Example response


{
    "username": "asterki",
    "count": 2,
    "log": [
        {
            "username": "asterki",
            "description": "test",
            "duration": 60,
            "date": "2016-03-23"
        },
        {
            "username": "asterki",
            "description": "test 2",
            "duration": 60,
            "date": "2016-03-24"
        }
    ]
}
